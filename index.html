Bay Management

Control how trolleys are staged in physical bays before dispatch.

The Driver View shows bay assignments and a trolley diagram per run showing which trolleys belong to each customer.
All orders on the same van share one bay. Bays are set per van in the Driver View. Pickers are prompted to select a bay when marking an order as Ready for Delivery. Bay numbers appear in the order modal and the bay board groups orders by bay.
Total physical bays available. Default: 3.
Set a maximum trolley limit for each bay individually. Excess trolleys can be split to another bay.

ETA Calculation

Configure how estimated delivery times are calculated and shown to customers.

Earliest a van departs the warehouse. Default: 07:45.
Latest a delivery can occur. Default: 16:45.
Time to pick one trolley of plants. Default: 60.
Time to load the van at warehouse before departure. Default: 20.
Unload + handover time per customer. Default: 10.
Extra time added when Barcoded Labels are required. Default: 15.
Extra time added when Pre-Priced Labels are required. Default: 20.
When enabled, orders that can't be fulfilled today show their next available slot (purple badge, +1d) instead of a red warning.
When enabled, customers whose picking hasn't started yet show no ETA instead of a rough estimate. When disabled, a grey "est." ETA is shown based on trolley count.
